# NoteFlow

App de productividad móvil construida con React Native y Expo.
Permite gestionar tres tipos de contenido: notas, tareas e ideas.

## Stack tecnológico
- React Native + Expo
- TypeScript
- Expo Router
- Gluestack UI
- Zustand
- AsyncStorage
