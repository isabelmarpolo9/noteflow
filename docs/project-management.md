# Gestión del proyecto

## Herramienta utilizada
Trello: https://trello.com/b/plv66xjs/noteflow

## Estructura del tablero
El tablero está dividido en 5 columnas:

- **Backlog:** funcionalidades definidas pero no empezadas
- **Todo:** funcionalidades listas para empezar
- **In Progress:** funcionalidades en desarrollo activo
- **Review:** funcionalidades terminadas pendientes de revisión
- **Done:** funcionalidades completadas y verificadas

## Tarjetas
Cada funcionalidad principal de NoteFlow tiene su propia tarjeta
con un checklist de subtareas técnicas concretas.
A medida que avance el desarrollo, las tarjetas se moverán
de columna según su estado.
